# P4 グランドクロス練習

FF14「絶妖星乱舞」P4 グランドクロスの判断練習ページです。

デバフ付与、無の氾濫、デバフ処理、マジックチャージ、P4通しの流れを、ロール別に確認・練習できます。

公開ページ: https://pilsnerdrinker.github.io/practice-GC/

## 練習できること

- 1回目グランドクロスのデバフ判断
- 2回目グランドクロスのデバフ判断
- 3回目グランドクロスのアラガン/超越・傷・ビーム判断
- 無の氾濫のビーム位置判断
- 早/遅デバフ処理の位置確認
- マジックチャージとマジックアウトの安置判断
- P4通しの連続判断

## モード

### ギミック理解

デバフや真偽を見て、処理内容を選ぶモードです。

- 1回目グランドクロス
- 2回目グランドクロス
- 3回目グランドクロス
- マジックチャージ

### 位置確認

マップ上で安置や処理位置を選ぶモードです。

- 無の氾濫
- デバフ処理
- マジックチャージ
- P4通し練習

### 実践練習

制限時間つきで連続判断するモードです。

デフォルトでは「実践練習 / P4通し練習」が開きます。

P4通し練習は13枚構成です。

1. 1回目GC読取
2. 2回目GC読取
3. 3回目GC読取
4. 無の氾濫
5. 早デバフ処理
6. 早視線処理
7. 中央集合
8. 早ほのお処理
9. 遅雷水処理
10. 遅視線処理
11. 中央集合
12. 遅つなみ処理
13. マジックアウト

## 操作

- 自分ロールを選ぶと、そのロール視点の問題になります。
- パーティーリストのロールをクリックしても、自分ロールを変更できます。
- 「真偽を表示」で、パーティーリスト上の真偽マークを切り替えます。
- 「詳細表示」を外すと、デバフ表示が画像アイコン表示になります。
- 「別の問題」で問題セットを更新します。
- 「前の問題」で直前の問題セットに戻ります。
- 実践練習では「スタート」で開始し、「一時停止」で停止/再開できます。
- 実践練習の「一問ごと」を外すと、結果表示後に次の問題セットへ自動で進みます。

## デバフ秒数について

パーティーリストの秒数は、デバフを読むための目安表示です。

この練習ページでは、回答表示やデバフ表示ディレイを入れているため、実戦の秒数をリアルタイムで完全再現しているわけではありません。

- `1m` は残り1分以上の目安です。
- 59秒以下の数字は、早/遅や処理順を読むための目安です。
- 3回目GCのデバフは、簡易アイコン表示時に `15` を表示します。
- 詳細表示では、3回目GCデバフは `死者の傷` などの名前だけを表示します。
- 実践通しでは、処理済みのデバフは非表示になります。
