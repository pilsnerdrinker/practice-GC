# P4 グランドクロス練習

FF14「絶妖星乱舞」P4 グランドクロスの読取と位置取りを練習するためのページです。

デバフ、真偽、カオスのほのお/つなみ、マジックチャージ、無の氾濫をランダムに出題し、自分のロールでどこへ行くかを確認できます。

## できること

- 1回目グランドクロス、2回目グランドクロス、3回目グランドクロス、マジックチャージのギミック理解
- 無の氾濫、デバフ処理、マジックチャージ、P4通し練習の位置確認
- 制限時間つきの実践練習
- 自分ロールの切り替え
- パーティーリスト上の真偽表示、詳細表示の切り替え
- 正解表示、前の問題、別の問題、成績リセット

## モード

### ギミック理解

パーティーリストに表示されたデバフと真偽を見て、選択肢で判断するモードです。

- 1回目グランドクロス: 水圧縮、フォークライトニング、早叫声、加速度爆弾、ほのお/つなみを読む
- 2回目グランドクロス: 1回目デバフの残り時間と2回目デバフをまとめて読む
- 3回目グランドクロス: 死の超越、アラガンフィールド、青/紫ビーム、TH/DPS側を読む
- マジックチャージ: 雷床、氷床、マジックアウトの真偽を順番に読む

### 位置確認

マップをクリックして、実際の移動先を確認するモードです。

- 無の氾濫
- デバフ処理8枚
- マジックチャージ3枚
- P4通し13枚

### 実践練習

3カウント後にタイマーが始まり、制限時間内に回答するモードです。
時間切れは不正解として扱い、正解を表示して次へ進みます。

## 主な判定

| デバフ・要素 | 真 | 偽 |
| --- | --- | --- |
| 水圧縮 | 頭割り | 散開 |
| フォークライトニング | 散開 | 頭割り |
| 加速度爆弾 | 動かない | 動く |
| 呪詛の叫声 | 見ない | 見る |
| 混沌の炎（ほのお） | タケノコ円 | タケノコドーナツ範囲 |
| 混沌の水（つなみ） | タケノコドーナツ範囲 | タケノコ円 |
| 死の超越 | 同色ビーム | 逆色ビーム |
| アラガンフィールド | 逆色ビーム | 同色ビーム |

## メモ

- 1回目GCの水圧縮・フォークライトニングの早遅は共通です。
- 2回目GCの水圧縮・フォークライトニングの早遅は、1回目GCと逆になります。
- 1回目の叫声は早視線、2回目の叫声は遅視線です。
- 2回目GCでは、2回目に付与されたデバフに下線と枠を付けて表示します。
- 2回目の早ほのお/つなみは45秒、遅ほのお/つなみは1m表示です。
- マジックアウトでは、チャージ時に覚えた「踏む/踏まない」の判断に、もう一度真偽をかけます。
